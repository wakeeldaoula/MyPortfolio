# Resume File

Please add your resume PDF file here as `resume.pdf`.

This file will be served when users click the "Download Resume" button on your portfolio website.

## How to add your resume:

1. Save your resume as a PDF file
2. Rename it to `resume.pdf`
3. Place it in this `public` folder
4. The download button will automatically work

## Tips for your resume:

- Keep it to 1-2 pages
- Use a clean, professional format
- Include your contact information
- Highlight your data science projects
- List your technical skills (Python, SQL, ML, etc.)
- Include relevant coursework and certifications
