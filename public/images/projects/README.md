## Recommended Image Specifications

- **Size**: 800x450 pixels (16:9 aspect ratio)
- **Format**: PNG or JPG
- **Quality**: Optimized for web (under 500KB per image)

## Tips for Great Project Images

1. Take a screenshot of your project's main interface
2. Include key features visible in the shot
3. Use a clean, professional view
4. Crop unnecessary browser toolbars

If an image is missing or fails to load, the emoji fallback will be displayed instead.
